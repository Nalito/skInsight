## Initial Setup:
Clone repo and create a virtual environment
```
$ git clone https://github.com/Nalito/skin-defect-detection.git
$ cd skin-defect-detection
$ python3 -m venv venv
$ . venv/bin/activate
```
Install dependencies
```
$ (venv) pip install -r requirements.txt
```
